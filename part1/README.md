Part 1: Data Acquisition, Cleaning and Exploratory Analysis  

The aim of this project is to clean the Medical Insurance dataset and conduct exploratory data analysis prior to applying machine learning techniques.

Dataset  
Dataset Name: Medical Insurance Cost Dataset  
Number of Rows: 1338  
Number of Columns: 7  
Target Variable: charges  

Libraries Used  
1) pandas  
2) numpy  
3) matplotlib  
4) seaborn  
5) scikit-learn  

Files  
1) part1.ipynb  
2) cleaned_data.csv  
3) insurance.csv  
4) plots  

How to Run  
1) Use the requirements.txt file to install all required libraries.
2) Open the Jupyter notebook or Google Colab file named part1.ipynb.
3) Execute the code cells in order, starting from the first cell to the last.
4) A file named cleaned_data.csv will be generated automatically.

Description  
In this part of the project, the dataset was cleaned.
Missing values were addressed using median imputation, duplicate entries were removed, data types were adjusted for efficiency, visualizations were created, and the cleaned dataset was saved for use in Part 2.
