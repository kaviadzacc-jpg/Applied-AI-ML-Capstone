Part 1: Data Acquisition, Cleaning and Exploratory Analysis

Objective

The objective of this project is to clean the Medical Insurance dataset and perform exploratory data analysis before machine learning.

Dataset

Dataset Name:
Medical Insurance Cost Dataset

Rows:
1338

Columns:
7

Target:
charges

Libraries Used

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

Files

- part1.ipynb
- cleaned_data.csv
- insurance.csv
- plots

How to Run

1. Install the required libraries using requirements.txt.
2. Open part1.ipynb in Jupyter Notebook or Google Colab.
3. Run all cells from top to bottom.
4. cleaned_data.csv will be generated automatically.

Description 

The dataset was cleaned, missing values were handled using median imputation, duplicate rows were removed, data types were optimized, visualizations were created, and the cleaned dataset was saved for use in Part 2.
