Part 1: Data Acquisition, Cleaning and Exploratory Analysis

The objective of this project is to clean the Medical Insurance dataset and perform exploratory data analysis before machine learning.

Dataset
Dataset Name: Medical Insurance Cost Dataset
Rows: 1338
Columns: 7
Target: charges
Libraries Used
1) pandas
2) numpy
3) matplotlib
4) seaborn
5) scikit-learn
Files
1) part1.ipynb
2) cleaned_data.csv
3) insurance.csv
4) plots
How to Run
1) Use the requirements.txt to install libraries.
2) View Jupyter notebook or google colab file part1.ipynb
3) Execute every cell sequentially from the starting to the last cell.
4) A file named cleaned_data.csv will be created automatically.
Description
The dataset was cleaned in this part, missing values were handled using median imputation, duplicate rows were dropped, datatypes were optimized, visualizations were created, and the cleaned dataset was saved.
